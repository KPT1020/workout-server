## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/KPT1020/workout-server.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run the app
   ```sh
   npm start
   ```
